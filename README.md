# ArchiWeb
Лендинг создан по макету Figma, адаптивная версия создана самостоятельно, без макета.


## Функционал
- Hover - состояние для элементов с которыми может взаимодействовать пользователь.
- Активная карта в подвале сайта.


## Технологии
- CSS.
- HTML.
- Flexbox.
- Grid - сетка.
- БЭМ
